{\fontsize{14}{16}\selectfont \textbf{Abstract}}

\vspace{1cm}

\justifying
\emph{This project presents a single-object tracking system based on the SORT framework and adapted for a hardware-oriented implementation: the pipeline uses foreground subtraction to extract a bounding box from each video frame, converts the detection into a SORT-compatible state, and applies constant-velocity prediction with IoU-based gating for update decisions, while the state, covariance, and noise terms are represented using compact fixed-point formats and vectorized storage to support a Verilog controller; validation was performed by comparing Python SORT reference tracks (`tracks_sort_reference.csv`) to the custom hardware output (`detections_hw.csv`) using the `Video/compare_trackers.py` script (format `cxcyhr`), yielding a mean IoU of 0.9429 and mean center error of 1.05 px over 326 shared frames, which demonstrates close alignment between the hardware output and the traditional SORT implementation, while an extended parameter sweep highlights sensitivity to process and measurement noise settings.}

\vspace{0.75em}

\noindent \emph{\textbf{Keywords:} Object tracking; SORT; Kalman filter; Fixed-point; 
Hardware acceleration; IoU}
